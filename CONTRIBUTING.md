# 贡献指南

感谢你考虑为智能测试用例生成器做出贡献！

## 如何贡献

### 1. Fork 本仓库

点击仓库页面右上角的 "Fork" 按钮。

### 2. 克隆你的 Fork

```bash
git clone https://github.com/yourusername/ai-testcase-generator.git
cd ai-testcase-generator
```

### 3. 创建功能分支

```bash
git checkout -b feature/your-feature-name
```

### 4. 进行修改

- 改进现有prompt模板
- 添加新的测试场景
- 修复bug
- 完善文档

### 5. 提交修改

```bash
git add .
git commit -m "Add: 描述你的修改"
```

### 6. 推送到你的 Fork

```bash
git push origin feature/your-feature-name
```

### 7. 创建 Pull Request

在 GitHub 上创建 Pull Request，描述你的修改内容。

## Pull Request 指南

- 📝 PR 标题要清晰明了
- 📋 详细描述你的修改内容和动机
- ✅ 确保代码格式规范
- 🧪 如果添加了新功能，请附带测试用例
- 📚 更新相关文档

## 报告问题

如果你发现了bug或有新功能建议，请：

1. 先搜索已有的 issues
2. 创建新的 issue
3. 使用模板（如果可用）
4. 提供详细的问题描述和复现步骤

## 代码规范

- 使用清晰的中文命名
- 代码注释使用中文或英文
- 遵循现有代码风格
- 保持代码简洁易读

## 许可证

通过贡献代码，你同意你的贡献将遵循 MIT 许可证。

感谢你的贡献！ 🙏
