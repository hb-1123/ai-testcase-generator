# 智能测试用例生成器
## AI Test Case Generator

## Overview
智能测试用例生成器，支持多种测试场景和格式输出。采用模块化设计，提供基础版、专业版、接口测试、UI测试等多种生成模式。

## Capabilities
- 自动生成标准格式测试用例
- 支持多种测试设计方法（边界值、等价类、场景分析等）
- 多格式输出（Markdown/JSON/YAML/CSV）
- 智能推断测试场景和边界条件
- 自动匹输出语言
- 灵活的优先级定义（P0/P1/P2/P3）

## Prompts
- **default**: 基础测试用例生成（通用场景）
- **professional**: 专业版测试用例生成（高级测试设计方法）
- **api**: 接口测试用例生成（API/后端服务）
- **ui**: UI/界面测试用例生成

## Usage
```
生成一个用户登录功能的测试用例
```
```
使用边界值分析法，为年龄输入框生成测试用例，输出为JSON格式
```
```
为一个订单创建API生成接口测试用例，包含正向、异常、边界测试
```

## Notes
- 默认输出语言自动匹配输入语言
- 测试用例数量根据需求复杂度自动判断
- 支持中途修改输出格式
- 提供正向、负向、边界、异常等全面场景覆盖
