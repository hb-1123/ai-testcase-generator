# 智能测试用例生成器

## AI Test Case Generator

一个基于AI智能的测试用例生成工具，支持多种测试场景和格式输出。

## 📖 简介

智能测试用例生成器是一款专注于帮助测试工程师快速生成高质量测试用例的AI工具。它采用模块化设计，提供基础版、专业版、接口测试、UI测试等多种生成模式，满足不同测试场景的需求。

**广泛适用性**：本工具适用于各行各业的测试需求，无论是金融、电商、医疗、教育还是其他行业，都可以根据个性化业务需求生成针对性的测试用例。

## 🚀 使用方法

### 使用方式

1. 克隆仓库：
```bash
git clone https://github.com/hb-1123/ai-testcase-generator.git
```

2. 打开对应的prompt文件（如 `prompts/default.md`）

3. 复制prompt内容到AI对话系统中

4. 按照prompt中的说明描述你的测试需求

## 📊 输出格式

### Markdown表格（默认）

```markdown
| 用例ID | 用例标题 | 前置条件 | 测试步骤 | 预期结果 | 优先级 |
|--------|----------|----------|----------|----------|--------|
| TC-001 | 用户正常登录 | 用户已注册 | 1. 输入账号密码<br>2. 点击登录 | 登录成功，跳转首页 | P0 |
```

### JSON格式

```json
{
  "testCases": [
    {
      "id": "TC-001",
      "title": "用户正常登录",
      "precondition": "用户已注册",
      "steps": ["输入账号密码", "点击登录"],
      "expectedResult": "登录成功，跳转首页",
      "priority": "P0"
    }
  ]
}
```

## 📋 优先级定义

- **P0**：核心功能冒烟测试用例，必须通过
- **P1**：重要功能测试用例，通常需要执行
- **P2**：一般功能测试用例，可以延后执行
- **P3**：次要功能或边界测试用例，低优先级
